# Personal Website

This repository contains the code for Addisyn Bordthauser's personal website.

## Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone <repository_url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd mis_project
    ```
3.  **Open the website:**
    Simply open `index.html` in your preferred web browser to view the website.

## Project Structure

-   `index.html`: Home Page
-   `hobbies.html`: Hobbies Page
-   `discover.html`: Discover Page (UMD & Duluth)
-   `resume.html`: Resume Page & Career Interests Section
-   `game.html`: Interactive Game Page
-   `style.css`: Global CSS styles for the website
-   `media/`: Directory for images, videos, and other media files.

## Customization

-   **Content:** Update the HTML files (`.html`) with your specific text and details.
-   **Images/Videos:** Replace placeholder images and videos in the `media/` directory with your actual files. Ensure they are optimized for web.
-   **Styling:** Modify `style.css` to adjust colors, fonts, and other design elements to your liking.

## Features

-   **Responsive Design:** The website is designed to be responsive and work well on both desktop and mobile devices.
-   **Navigation Menu:** A consistent navigation menu is provided across all pages.
-   **Professional Presentation:** Clean and organized code with a professional aesthetic.
